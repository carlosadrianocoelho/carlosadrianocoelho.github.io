INTRODUÇÃO AO GOOGLE APPS SCRIPT

17 de Janeiro de 2019

A Google disponibiliza vários serviços gratuitos aos seus usuários, como o Gmail (serviço de email), GDrive (serviço de armazenamento de arquivos em nuvem), GSheets (aplicativo de planilhas), dentre outros. Esses aplicativos fazem parte da plataforma GSuite e quem os usa sabe o quanto eles facilitam a nossa vida, e podem facilitar ainda mais pois é possível utilizar a linguagem de programação apps script para personalizar esses serviços.

A linguagem apps script é baseada na linguagem JavaScript, é bem leve e pode ser usada para criar aplicativos web que ficam armazenados nos servidores da Google. Além disso, o apps script possui um ambiente online onde é possível editar, testar e publicar o seu código, que pode ser acessado aqui. Para acessar o editor é necessário possuir uma conta Google.

Com o domínio dessa linguagem é possível criar inúmeras aplicações, e também pode-se integrar os serviços com apps escritos em outras linguagens e que rodam em outras plataformas. A primeira aplicação que eu pensei em fazer quando eu conheci a linguagem apps script foi a de um banco de dados utilizando GSheets. Utilizando-se as planilhas que estão online o tempo todo voce pode inserir e recuperar informações a partir de um aplicativo android, iOS ou até mesmo do seu site pessoal. Uma outra ideia interessante pode ser a de usar o GDrive para armazenar fotos, vídeos e outros arquivos, e poder visualizá-los a partir de um outro aplicativo de outra plataforma. É claro que existem limites, mas utilizar os apps da plataforma GSuite é uma forma excelente de escrever aplicações de alto nível sem gastar nada, principalmente para quem está começando e não tem muitos recursos disponíveis.
