O primeiro script disponibilizado mostra um código que cria um documento (Google Doc) no GDrive (Google Drive) e envia um email para o Gmail com um link que permite acessar esse documento.

Passo 1:
Acesse https://script.google.com/ para abrir o editor de script e click em Start Scripting. É preciso ter uma conta Google (crie uma se não tiver ainda).

Passo 2:
Uma tela pergunta que tipo de projeto se quer criar. Clique em projeto em branco (blank project).

Passo 3:
Delete o código gerado automaticamente, copie o código abaixo e cole no editor do Apps Script. 

/**
 * Creates a Google Doc and sends an email to the current user with a link to the doc.
 */
function createAndSendDocument() {
  // Create a new Google Doc named 'Hello, world!'
  var doc = DocumentApp.create('Hello, world!');

  // Access the body of the document, then add a paragraph.
  doc.getBody().appendParagraph('This document was created by Google Apps Script.');

  // Get the URL of the document.
  var url = doc.getUrl();

  // Get the email address of the active user - that's you.
  var email = Session.getActiveUser().getEmail();

  // Get the name of the document to use as an email subject line.
  var subject = doc.getName();

  // Append a new string to the "url" variable to use as an email body.
  var body = 'Link to your doc: ' + url;

  // Send yourself an email with a link to the document.
  GmailApp.sendEmail(email, subject, body);
}

Passo 4:
Vá ao menu e clique em Arquivo->Salvar, nomeie seu arquivo e clique em OK.

Passo 5:
Para executar o script vá ao menu e clique em Run->CreateandSendDocument. A execucção do script requer a sua autorização, clique OK para dar premissão.

Passo 6:
Depois que o script terminar de executar, abra o gmail e veja que recebeu um novo email com um link(o link do novo documento). Por fim, abra o GDrive e veja que tem dois arquivos novos, o do documento e o do script.

Esse exemplo mostra que a linguagem apps script é muito útil e vale a pena aprendê-la. Para ver a postagem oficial, acesse o artigo Overview of Apps Script.
